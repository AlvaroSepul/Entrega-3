# Telecom X - Predicción de Cancelación de Clientes

## Descripción
Este proyecto analiza la cancelación de clientes (churn) en Telecom X usando Python.  
El trabajo incluye limpieza de datos, análisis exploratorio y construcción de modelos predictivos para identificar factores asociados a la evasión de clientes.

## Objetivo
Entender qué variables están más relacionadas con la cancelación del servicio y generar una base para futuras estrategias de retención.

## Contenido del proyecto
- Carga de datos desde archivo JSON / CSV
- Limpieza y tratamiento de datos
- Creación de nuevas variables
- Estandarización de columnas
- Análisis descriptivo y exploratorio
- Preparación de datos para machine learning
- Entrenamiento de modelos predictivos
- Evaluación de modelos
- Análisis de variables más relevantes

## Herramientas utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Modelos utilizados
- Regresión Logística
- Random Forest

## Principales resultados
A partir del análisis se identificó que la cancelación de clientes está relacionada con variables como la antigüedad del cliente, el tipo de contrato y el nivel de gasto.  
Los modelos permitieron detectar patrones útiles para anticipar qué clientes tienen mayor riesgo de cancelar.
